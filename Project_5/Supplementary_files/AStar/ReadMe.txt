Instructions on running the program:

1. Open maze_map.ttt in V-rep
2. Change position of Turtlebot to change start position, if desired.
2. Start Simulation
3. Open and Run MazeVREP.m in MATLAB
4. Select goal point position when map of the configuration space pops up
5. After path is shown on map, go to V-rep to see the robot simulation
